# Access Transport Company Website

A modern, responsive, and accessible website for Access Transport Company, built from scratch with CSS and JavaScript. Features a green brand color (#3DB635), animated UI, and interactive effects.

## Features
- Home: Hero section with vehicle image, headline, tagline, animated buttons, and scroll effects
- About: Mission, values, and story with reveal animations
- Services: Cards/table for delivery services and rates, with hover animations
- Contact: Functional contact form with animated inputs and submit button
- Fully responsive, accessible, and visually polished

## Tech Stack
- HTML5, CSS3 (custom, no frameworks), JavaScript (vanilla)

## Getting Started
Open index.html in your browser or deploy to any static web host.
